## 主要项目：

- 重要事项

## 总结流程：

**1.一天两次填写：**

>早上一次，开始计划；
>
>晚上一次，重新总结

**2.日期：日期+星期**

**3.待完成事项：**

>前一天填写“待完成事项”；
>
>后一天填写“当天需要完成事项”

**4.完成事项：当天总结时填写当天完成情况**
